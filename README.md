# 🛍️ E-commerce Website with AI-Inspired Outfit Builder

A modern **frontend e-commerce website** that integrates an **AI-inspired outfit recommendation system** to enhance user shopping experience.

---

## ✨ Features

* 🧠 **AI Outfit Builder**
  Recommends outfits based on **gender** and **occasion**

* 🛒 **Shopping Cart System**

  * Add to Cart
  * Remove Items
  * Total Price Calculation
  * Uses **localStorage**

* 🧾 **Product Categories**

  * Men's Wear
  * Women's Wear
  * Kid's Wear
  * Shoes

* 🔐 **Authentication UI**

  * Login Page
  * Signup Page
  * Show/Hide Password

* 📱 **Fully Responsive Design**

  * Mobile-friendly layout
  * Optimized for all screen sizes

* 🎨 **Modern UI Design**

  * Clean layout
  * Hover effects & animations
  * Deal pricing display

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Responsive + Flexbox + Grid)**
* **JavaScript (Vanilla JS)**
* 
---

## ⚙️ How It Works

1. User browses products on homepage
2. Clicks **Add to Cart**
3. Items stored in **localStorage**
4. Cart page displays:

   * Product image
   * Description
   * Price
5. AI Builder suggests outfits dynamically
