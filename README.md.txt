# 🛍️ E-commerce Website with AI-Inspired Outfit Builder

A modern frontend e-commerce website with AI-based outfit recommendations.

## 🚀 Features
- AI Outfit Builder (based on gender & occasion)
- Add to Cart system (localStorage)
- Responsive design (mobile-friendly)
- Login & Signup UI
- Product categories (Men, Women, Kids, Shoes)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Future Improvements
- Backend integration
- Payment gateway
- User authentication
- Wishlist feature
